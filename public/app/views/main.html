Introduction

This is a demo application that walks through what's required to talk to the following AWS services using node.js, AngularJS, and socket.io:
  • DynamoDB - A simple dataset backed by DynamoDB
  • RDS - The same fields as the Dynamo example, but using RDS
  • S3 Dynamo - Upload a file to S3, keep track with DynamoDB
  • S3 RDS - Upload a file to S3, keep track with RDS
  • SES - Email a user!

A few SNS bits are covered in the server code as well.

Start with the DynamoDB example, and enter an email address and a name. Next, you can go to the S3 Dynamo example and upload an image or two. Finally, if you visit the SES page, you can send yourself an email with your media.
